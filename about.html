About the Phanatic Stats Project

This project is a multi-page web application built for INFO151 to demonstrate our understanding of HTML, CSS, and JavaScript using a Phillies-themed stats hub. The site includes a roster page with player cards, summary stats, and forms for contact and subscription.

Our goal was to organize content using a clear navigation structure, reusable components, and an organized project folder, while also showcasing interactive elements such as hover effects, computed stats, and basic form validation.
